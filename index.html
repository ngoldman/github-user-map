github-user-map

A simple app that looks up a github user and shows their approximate location on a map.

Adapted from ReactJS: GitHub User Cards.

I ended up making two versions:

React

Made during a MaptimePDX meetup led by jgravois.

hyperx

Made to compare a hyperx/main-loop implementation with react. Way less tooling headaches.
